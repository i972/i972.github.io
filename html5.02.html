1. HTML Tag 출력하기

HTML 은 꺽쇠 < > 안에 들어 있는 것을 기본적으로
내용이 아닌 '태그'로 해석하기 때문이다.
그래서 &문자를 이용해서 특수문장부호를 표현 해줘야 한다. 


          &nbsp;      # ” ” : 공백(스페이스 한 칸)을 의미
          &lt;        # 부등호(<) 
          &gt;        # 부등호(>) 
          &amp;       # 앰퍼샌드(&) 기호 
          &quot;      # 쌍따옴표(“) 
          &#035;      # sharp(#) 
          &#039;      # 따옴표(‘)
