1. 리눅스 SSH 로 GUI 실행하기

원격 서버의 X11 Forwarding 설정을 하고, 접속할때는 -X 옵션을 사용 하자

< X11 Forwarding 설정 가이드 >



          # X11 server mode 설정 (/etc/ssh/sshd_config 설정)
          # sshd_config 을 열어서 아래 3줄을 추가(주석 표시 제거

          vi /etc/ssh/sshd_config

              X11Forwarding yes
              X11DisplayOffset 10
              X11UseLocalhost yes 

          # ssh 재시작
          sudo systemctl restart sshd_config

          # 서버 접속할때 -X 옵션을 이용하자
          ssh -X i972@192.168.0.2

          # 실행 결과 확인, 제대로 실행된다면 ok
          xclock
