1. HTML 파비콘(주소창 아이콘) 적용 방법

웹 페이지에서 주소창 위에 나타나는 아이콘(파비콘)

  • 파비콘 파일 제작
  • HTML 파일 적용


        ** 파비콘 제작
        # 파비콘으로 사용할 이미지 파일을 준비
        # www.favicon-generator.org/
        # 파일 선택 -> Generate only 16x16 favicon.ico 선택 -> Create Favicon 클릭
        # 바뀐 화면에서 Download the generated favicon 클릭후 저장 

        ** HTML 코드 적용
        # <head></head>사이에 다음 코드 입력
        <link rel="icon" href="파일주소" type="image/x-icon" sizes="16x16">

        ** .ico 로 바꿀필요 없이 .png 파일로 바로 적용 가능하다 - 수정 2022.10.14
        <link rel="icon" type="image/png" sizes="16x16" href="images/cool.png" />
