1. Docker 호스트와 컨테이너의 파일시스템 연결 -v

컨테이너와 호스트의 파일시스템을 연결
호스트의 파일시스템을 통해 변경된 내용이 컨테이너의 파일시스템에
자동으로 연동되게 만들어 보자 



        # volume 을 통해 파일 시스템 연결 -v
        $ docker run -v ~/문서/docker/htdocs:/usr/local/apache2/htdocs httpd

        # 포트, 이름, 볼륨을 한번에...
        $ docker run --name ws2 -p 7778:80 -v ~/문서/docker/htdocs:/usr/local/apache2/htdocs httpd

        # sshfs 를 이용한 원격 접속
        # sshfs [리모트서버ip]:[remote 디렉토리][local 디렉토리]
        $ sshfs hp:~/문서/docker/htdocs ~/문서/docker/htdocs
