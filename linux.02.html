1. 리눅스 원격 디렉토리 마운트 SSHFS

SSH 파일 시스템(SSH Filesystem)은 SSH 연결을 통해 원격 디렉터리를 마운트하기 위한 FUSE 기반 파일 시스템 클라이언트, SFTP 프로토콜을 사용하고 있으며 대부분의 SSH 서버에서 기본적으로 사용하도록 설정되어 있어 서버 측에서 추가 구성을 할 필요가 없는게 장점이다

< SSHFS 설치 >



          sudo apt install sshfs

< SSHFS 원격 파일 시스템 마운트 >



          # user@host : 접속할 서버의 사용자 ID @ 접속할 서버의 주소
          # remote_derectory : 마운트할 서버의 디렉토리  
          # mountpoint : 마운트할 로컬 디렉

          sshfs [user@]host:[remote_directory] mountpoint [options]
          sshfs i972@192.168.0.2:/home/i972 ~/문서/hp_envy_6

          # hp 는 ./ssh/config 파일에 정의 되어 있다
          sshfs hp:/home/i972 ~/문서/hp_envy_6

< SSHFS 자동 마운트 >



          # 원격 디렉토리를 영구적으로 마운트하려면 로컬 시스템의 /etc/fstab 
          # 파일을 편집하고 새 마운트 항목을 추가
          # 이렇게 하면 시스템이 부팅될 때 원격 디렉터리가 자동으로 마운트 된다
 
          vi /etc/fstab

          # /remote/dir : 접속할 서버의 디렉토리
          # /local/mountpoint : 로컬 디렉토리
          # fuse.sshfs : 파일 시스템 유형

          user@host:/remote/dir  /local/mountpoint  fuse.sshfs  defaults  0  0

< SSHFS 마운트 확인 및 헤제 >



          # 마운트 확인
          df -h

          # 마운트된 파일 시스템을 분리 하려면 umount, fusermount
          # 명령 뒤에 마운트된 디렉터리(mount point)를 지정 하면 된다

          umount /local/mountpoint
          fusermount -u /local/mountpoint
