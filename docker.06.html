1. Docker 명령어 실행 exec

컨테이너 안으로 들어 가서 container 를 수정해 보자



        # 컨테이너 내부의 명령을 실행해 보자 - exec
        # docker exec [컨테이너이름] [명령어]
        $ docker exec ws1 pwd

        # 컨테이너에 계속 머물면서 명령을 실행 - exec -it
        $ docker exec -it ws1 /bin/bash

        # 컨테이너 탈출 - exit
        $ exit
