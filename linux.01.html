1. 리눅스 Font 삭제

직접 설치한 폰트는 ~/.local/share/fonts 에 설치 된다. 



          # 직접 설치한 폰트 위치로 이동  ~/.local/share/fonts
          cd ~/.local/share/fonts

          # 원하는 폰트 삭제           
          rm Sunflower-Light.ttf

          # 글꼴 캐쉬를 업데이트
          fc-cache -fv
