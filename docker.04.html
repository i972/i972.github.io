Docker





1. Docker 컨테이너 run

하나의 image 는 여러개의 container 를 만들 수 있다.



        # container 생성
        $ docker run httpd

        # container 생성 - --name 이름추가
        $ docker run --name ws2 httpd

        # container 확인 - run
        $ docker ps

        # container 중지 - stop, 컨테이너가 삭제된 것은 아니다
        $ docker stop ws1

        # container 확인 - run & stop
        $ docker ps -a

        # container 재실행 - start
        $ docker start ws1

2. Docker container/image 삭제



        # container 삭제 - rm, 실행중인 컨데이너는 stop 한뒤에 삭제해야 된다
        $ docker rm ws1

        # container 삭제 - --force, 강제로 실행중인 컨테이너 삭제
        $ docker rm --force ws1

        # image 삭제- rmi, remove image
        $ docker rmi httpd
